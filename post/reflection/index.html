反射

08 Aug 2019, 09:32

概念

反射：把Java类中的各种结构（方法，属性，构造器，类名）映射成一个个的Java对象

代码

public class ReflectionTest {
    public static void main(String[] args) throws NoSuchMethodException, ClassNotFoundException, IllegalAccessException, InvocationTargetException, InstantiationException {

        /**
         * 3种方式获取Class对象
         */
        // 对象.getClass
        Class test01 = new Transportation().getClass();

        // 类.class
        Class test02 = Transportation.class;

        // Class.forName("包名.类名")
        Class test03 = Class.forName("Chapter_11_HttpServer.Reflection.Transportation");

        // 创建对象
        Transportation test = (Transportation)test03.getConstructor().newInstance();

    }

}

class Transportation{
    private String bus;
    private String car;
    private String helicopter;

    public Transportation() {
    }
}
