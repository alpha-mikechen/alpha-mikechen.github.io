Java数据结构与算法: Queue

16 Jul 2019, 07:36

循环队列

1. 实现理论

  • 建立一个大小为maxSize的数组
  • 定义头指针front指向头元素
  • 定义尾指针rear指向尾元素的后一位
  • 定义当数组中仅剩一个空闲单元时，队列已满。
  • front和rear的初始值均为0
  • 入队时，则在队尾添加元素，rear = （rear+1)%maxSize
  • 出队时，则移除队头元素，front = (front+1)%maxSize

注意： 实现循环的核心是：通过指针和数组大小的求余运算(%)，使数组下标循环。

2. 编程实现


class CircleArrayQueueClass{
    private int maxSize;
    private int front;
    private int rear;
    private int[] arr;

    public CircleArrayQueueClass(int maxSize) {
        this.maxSize = maxSize;
        front = 0;
        rear = 0;
        arr = new int[this.maxSize];
    }

    //判断队列是否为空
    public boolean isEmpty(){
        return front == rear;
    }

    //判断队列是否已满
    public boolean isFull(){
        return (rear+1)%maxSize == front;
    }

    //显示当前队列元素的个数
    public int size(){
        return (rear-front+maxSize)%maxSize;
    }

    //入队
    public void add(int data){
        if (isFull()){
            throw new RuntimeException("队列已满无法添加数据");
        }
        else {
            arr[rear] = data;
            rear = (rear+1)%maxSize;
        }
    }

    //出队
    public void remove(){
        if (isEmpty()){
            throw new RuntimeException("队列为空无法移除数据");
        }
        else {
            System.out.println("移除的数据是："+arr[front]);
            front = (front+1)%maxSize;
        }
    }

    //显示所有元素
    public void show(){
        if(isEmpty()){
            throw new RuntimeException("队列为空\n");
        }
        else {
            for(int i=front;i<front+size();i++){
                System.out.printf("arr[%d] = %d\n",(i)%maxSize,arr[(i)%maxSize]);
            }
        }
    }

    //显示头元素
    public void head() {
        if (isEmpty()) {
            throw new RuntimeException("队列为空");
        } else {
            System.out.println("队列头元素是："+arr[front]);
        }
    }
}
