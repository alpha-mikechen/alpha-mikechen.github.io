Java数据结构与算法: 5个单向链表的常见面试题

1. 求单链表中有效节点的个数，即不计算头节点

public int getLength(){
    if (headNode.next == null){
        return 0;
    }
    Node temp = headNode;
    int length = 0;
    while (temp.next!=null){
        length++;
        temp = temp.next;
    }
    return length;
}

2. 查找单链表中倒数第k个节点

public Node showLastIndex(int k){
    if (headNode.next == null){
        return null;
    }
    int length = getLength();
    int index = length-k;
    if (index<0 || index >length){
        System.out.println("越界");
        return null;
    }
    Node temp = headNode;
    for (int i=1;i<index;i++){
        temp = temp.next;
    }
    return temp.next;
}

3. 单链表的反转

思路

  • 定义一个辅助节点temp，帮助遍历原链表
  • 定义一个辅助节点tempNext，其指向temp节点的下一个，帮助摘取temp节点
  • 定义一个新的空节点reverseHead，每次遍历原链表时都顺序摘掉一个节点放在reverseHead有效节点的最前端
public static void reverse(Node headNode){
    //如果头节点为空，或者链表中只有一个有效节点，则无需反转
    if (headNode.next == null || headNode.next.next == null){
        return;
    }
    Node temp = headNode.next;
    Node tempNext = null;
    Node reverseHead = new Node(0,"","");
    while (temp!=null){
        tempNext = temp.next;
        temp.next = reverseHead.next;
        reverseHead.next = temp;
        temp = tempNext;
    }
    headNode.next = reverseHead.next;
}
